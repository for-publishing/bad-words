# bad-words
Список матных слов русского языка и морфемы для настройки логики фильтра матных слов. <br><br>

Слова закодированы в формате Hex \uXXXX. Их можно раскодировать на сайте https://crypt-online.ru/crypts/text2hex/ или подобном.
<br>
<br>

### Грубый список слов
В файле <b>gross-list</b> грубый список слов. По нему фильтр будет находить соответствие не со всеми матными словами, потому что много нестандартных.


### Логика вхождения
Можно настроить логику нахождения матных слов с помощю нескольких категорий вхождения.
Например:

- В файле <b>bad_word_if_contains</b> морфемы, которые содержатся в любой части слова.
- В файле <b>bad_word_if_exact</b> слова, которые точно соответствуют.
- В файле <b>bad_word_if_start</b> морфемы, с которых начинается слово.

По этой же логике добавлены исключения (<b>exception_if_contains</b>, <b>exception_if_exact</b>). Когда слово не матное.
