# bad-words
Список матных слов русского языка и морфемы для настройки логики фильтра матных слов.

Слова закодированы в формате Hex \uXXXX. Их можно раскодировать на сайте https://crypt-online.ru/crypts/text2hex/ или подобном.

В файле gross-list грубый список слов. По нему фильтр будет находить соответствие не со всеми матными словами, потому что много нестандартных.

Можно настроить логику нахождения матных слов с помощю нескольких категорий вхождения.
Например:

В файле bad_word_if_contains морфемы, которые содержатся в любой части слова.
В файле bad_word_if_exact слова, которые точно соответствуют.
В файле bad_word_if_start морфемы, с которых начинается слово.

По этой же логике добавлены исключения (exception_if_contains, exception_if_exact). Когда слово не матное.
